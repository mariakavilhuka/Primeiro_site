HTML

HTML abreviação para a expressão inglesa HyperText Markup Language, que significa: "Linguagem de Marcação de Hipertexto" é uma linguagem de marcação utilizada na construção de páginas na Web.

TAGS

As tags definem ao navegador a marcação de conteúdo.
As tags tem funcionalidades específicas. Sem o uso de tags o navegador exibe um texto plano.

As tags podem possuir atributos, ou seja, são parâmetros que definem algum tipo de informação
